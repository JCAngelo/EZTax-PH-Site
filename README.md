# EZTax Landing Page

A professional landing page for EZTax - an AI-powered tax compliance and bookkeeping solution for Filipino micro-enterprises and self-employed professionals.

## 🎯 Overview

EZTax simplifies Philippine tax compliance by enabling users to file BIR tax returns in just 4 steps using their smartphone. This landing page showcases the product features, target market, developer background, and cutting-edge technology to attract potential users.

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for CDN resources: fonts, icons, Chart.js)

### Installation

1. **Clone or download the repository**
   ```bash
   git clone <repository-url>
   cd Website-Bob
   ```

2. **Open the website**
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (http-server)
     npx http-server
     ```

3. **View in browser**
   - Navigate to `http://localhost:8000` (if using local server)
   - Or double-click `index.html`

## 🔧 Customization

### Updating Content

1. **Text Content**: Edit directly in `index.html`
2. **Styling**: Modify CSS variables in `styles.css` (lines 10-30)
3. **Colors**: Change gradient definitions in `:root` selector
4. **Revenue Data**: Update Chart.js data in `script.js` (lines 120-135)

### Adding New Sections

1. Add HTML structure in `index.html`
2. Add corresponding styles in `styles.css`
3. Add any interactive features in `script.js`
4. Update navigation menu if needed
